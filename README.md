# LULU-Compiler
A lexical and syntax compiler via Antlr:)

## Quick Start

```
git clone https://github.com/AMK9978/LULU-COMPILER.git
```

After that,you can run the file with Antlr plugin and test your input and export Java,JS ,... files.

## Built With

* Antlr 5.3
